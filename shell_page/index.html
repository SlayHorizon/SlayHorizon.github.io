Shell - Quelques commandes de bases:

pwd - print working directory - affiche le répertoire courant
- Affiche le chemin complet du répertoire courant dans le terminal.
- Exemple:
'pwd' == '/home/thomas/projet_web/'

cd - change directory - change de répertoire
- Change de répertoire courant.
- Exemples:
'cd sous_dossier_du_repertoire_courant/'
'cd ..' pour revenir en arrière.
'cd /home/thomas/Desktop' pour accéder au chemin complet.

man - manuel
- Accède au manuel d'une commande shell, de fonctiones en C etc.
- Attention plusieurs manuels peuvent être disponibles
- Exemples:
'man cd'.
'man 2 write' pour accéder à la fonction 'write' en C.

ls - list
- Liste les fichiers et dossiers du répertoire courant en les affichant dans le terminal.
- 'man ls' pour connaître les options.
- Exemple:
'ls --all' permet d'afficher les fichiers cachés avec un '.' au début.

mkdir - make directory
- Créer un dossier.
- Exemple:
'mkdir mon_nom_de_dossier'

touch
- Modifie le temps de modification d'un fichier mais peut aussi servir à en créer un.
- Exemple:
'touch journal.txt' créera un fichier nommé 'journal' avec l'extension '.txt'.

cat
- Affiche le countenu d'un fichier dans le terminal sous forme de texte.

mv - move - déplacer / renommer
- Déplacer OU renommer un fichier.
- Exemples:
'mv nom_du_fichier_actuel.txt nouveau_nom.txt' pour renommer.
'mv nom_du_fichier.txt mon_dossier/' pour déplacer.

gcc - gnu compiler collection
- Compile un programme à partir d'un fichier '.c'.
- Eexemple:
'gcc programme.c -o test', compilera programme.c
et donnera un éxécutable s'appelant test.
