Quelques commandes SHELL basiques:

pwd - print working directory - afficher le repertoire courant
:Elle affichera le chemin complet du repertoire courant dans le terminal.

cd - change directory - changer de repertoire
:Change de repertoire courant.
Exemple 'cd sous_dossier_du_repertoire_courant/'
'cd ..' pour revenir en arriere.
'cd /home/thomas/Desktop' accéder au chemin complet.

man - manuel
:Accéder au manuel d'une commande shell, de fonctiones en C etc.
Exemple 'man cd'.

ls - list
:Liste les fichiers et dossiers du repertoire courant en les affichant dans le terminal.
Cette commande possede beaucoup d'options, 'man ls' pour en savoir plus.
Exemple 'ls --all' permet d'afficher même les fichiers cachés avec un '.' au début!

mkdir - make directory - créer un dossier
:Créer un dossier, par exemple "mkdir mon_nom_de_dossier".

touch - créer un fichier
:Créer un fichier, par exemple "touch journal.txt" ou "touch mon_programme.c".
Beaucoup d'options utiles, 'man touch' pour en savoir plus.

mv - move - déplacer / renommer
:Déplacer OU renommer un fichier.
Exemple 'mv j09.txt j10.txt' pour renommer.
'mv j10.txt mon_dossier/' pour déplacer.

gcc - gnu compiler collection
:Compile un programme en C, peu permettre de récupérer un éxécutable.
Eexemple 'gcc programme.c -o test', compilera programme.c
et donnera un éxécutable s'appelant test.

GIT

git init
:initialise un git dans le dossier actuel.

git clone
:clone un repertoire git avec le nom de dossier voulu.
Eexemple 'git clone lien_du_repetoire mon_dossier'.

git add
:ajoute des fichiers à envoyer / modifier dans le repertoire git.
Exemple: 'git add mon_programme.c'

git commit
:Créer un commit rassemblant tous les modifications sauvegardées avec git add / rm.
Exemple 'git commit -m "J'ai ajouté tel fichier qui fait tel chose..."
'-m' permet d'ajouter un message à votre commit pour indiquer les changements que vous allez faire.

git push
:Envoie le commit pour sauvegarder les modifications et ajouts sur le répértoire git.
